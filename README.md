# starter-slidev

An opinionated starter for creating a presentation with [Slidev](https://sli.dev).

## Features

- Battery-included (formatter, git-hooks, Vercel config).
- Beautiful animated background effect.

## Note for users

I created this template for my personal use, so it may not be suitable for everyone. Feel free to fork and modify it to fit your needs.
